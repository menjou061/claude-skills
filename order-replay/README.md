# order-replay

简报生成后自动触发：拉取当日长桥订单，若有新操作则更新简报操作确认区并生成复盘文档。

## 触发时机

- 每日简报生成完毕后调用
- 也可手动触发：`/order-replay` 或 `order-replay 2026-05-20`

## 执行逻辑

```
1. 拉当日订单
   longbridge order --history --start <TODAY> --end <TODAY>

2. 判断是否有成交订单（Status = Filled）
   - 无成交 → 输出"今日无新操作，跳过"，结束
   - 有成交 → 继续

3. 更新简报操作确认区
   文件：01-财经/简报/每日财经新闻快讯.md
   找到当日简报末尾的「✅ 操作确认」表格
   - 计划内操作：对照表格逐行更新状态（✅已执行 / ❌未执行 / 🔄部分执行）
   - 计划外操作：在表格末尾追加新行，标注「⚠️ 计划外」

4. 生成复盘文档（仅当有计划外操作 或 有规则违反时）
   文件：01-财经/复盘/<DATE>_当日操作复盘.md
   内容：
   - 逐笔金铲铲打分（参照金铲铲策略文档规则）
   - 规则违反标注（14天间隔 / 止盈线 / 财报前不操作 / DCA遗漏）
   - 当日已实现盈亏估算
   - 未完成行动清单（若有遗漏的计划操作）
```

## 复盘文档命名

`<YYYY-MM-DD>_当日操作复盘.md`

## 金铲铲打分规则（快速参考）

| 评分 | 含义 |
|------|------|
| ⭐⭐⭐ | 完全符合框架，事后验证正确 |
| ⭐⭐ | 符合框架，执行合格 |
| ⭐ | 方向正确但流程有瑕疵 |
| ⚠️ | 计划外或流程偏差，但未违反铁律 |
| ❌ | 违反 P1/P2 规则（止盈线、间隔、财报前不操作等） |
| ❌❌ | 严重违规（止盈触发后反向操作、裸杠杆等） |

## 常见规则违反检查清单

- [ ] B6：同标的同方向操作间隔 < 14 天？
- [ ] 止盈线已触发但未执行清仓，或反向加仓？
- [ ] 财报当日/前夜追高加仓？
- [ ] SPY DCA 本周是否遗漏？
- [ ] 单笔操作是否超过总资金 5%（P1 生死线）？
- [ ] COIN/加密 beta 操作前是否先跑 `btc snapshot`？

## 输出示例

### 有操作时

```
📋 今日订单（2026-05-20）：3 笔成交

更新简报操作确认区：
✅ TSLA 清 2 股 @$406.20（计划内，止盈执行）
✅ DUOL 清零 1 股 @$114.50（计划内）
⚠️ NVDA 加仓 3 股 @$221.00（计划外，财报后追涨）

已生成复盘：01-财经/复盘/2026-05-20_当日操作复盘.md
```

### 无操作时

```
📋 今日订单（2026-05-20）：无成交，跳过。
```
